Experience

I have worked at Sun Microsystems, Global Crossing Conferencing, Boeing, and Motorola.

At Sun Microsystems, my team was responsible for globalizing and internationalizing the Sun.com website. I developed display renderers to retrieve localized content from a content managment sytem and display it on the website to company design specs. The display renderers utilized Java, Java Server Pages (JSP), XML, XSLT, and HTML and were developed on a UNIX system.

At Global Crossing Conferencing, I developed a server-side web application to moderate conference calls. Utilized Java, JSP, Servlets, XML, XSLT, HTML, SQL, and JDBC.

At The Boeing Company, I led a team of 3 developers to create an interactive maintenance manual for the Apache Helicopter. This manual was a custom SGML browser with hyperlinks, hotspotted CGM graphics, bookmarks, annotations, and amendment capability. The manual was written in Visual C++ with MFC and ran on Windows NT.

At Motorola, I was responsible for designing, managing, and implementing an inventory control and verification system using C++, object-oriented analysis and design, Oracle SQL, and Motif GUI libraries on a UNIX system
